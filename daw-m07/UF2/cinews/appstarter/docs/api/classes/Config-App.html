-------------------------------------------------------------------------- Cookie SameSite --------------------------------------------------------------------------

public string|null $cookieSameSite = 'Lax'

Configure cookie SameSite setting. Allowed values are:

  • None
  • Lax
  • Strict
  • ''

Alternatively, you can use the constant names:

  • Cookie::SAMESITE_NONE
  • Cookie::SAMESITE_LAX
  • Cookie::SAMESITE_STRICT

Defaults to Lax for compatibility with modern browsers. Setting '' (empty string) means default SameSite attribute set by browsers (Lax) will be set on cookies. If set to None, $cookieSecure must also be set.

If your server is behind a reverse proxy, you must whitelist the proxy IP addresses from which CodeIgniter should trust headers such as X-Forwarded-For or Client-IP in order to properly identify the visitor's IP address.
